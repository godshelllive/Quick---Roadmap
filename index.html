Guide d'utilisation

Bienvenue sur Quick - Roadmap

Quick - Roadmap est votre outil de gestion d'objectifs intelligent. Organisez vos projets, suivez vos progrès et collaborez efficacement avec votre équipe.

Gestion des projets

  • Créer un projet : Cliquez sur le nom du projet actuel pour ouvrir le sélecteur, puis "Nouveau Projet"
  • Protection par mot de passe : Sécurisez vos projets sensibles avec un mot de passe
  • Accès automatique : Les propriétaires de projet accèdent automatiquement sans mot de passe
  • Collaborateurs : Voyez qui travaille sur le projet en temps réel

Gestion des objectifs

  • Créer un objectif : Bouton "Nouvel Objectif" → Remplissez les informations
  • 3 états disponibles : En cours, Terminé, Suspendu
  • Dates intelligentes : Suivi automatique des délais et retards
  • Validation rapide : Cochez la case pour marquer comme terminé
  • Filtres pratiques : Triez par statut pour une vue claire

Système de commentaires et de chat

  • Commentaires collaboratifs : Discutez sur chaque objectif
  • Discussions avec d'autres utilisateurs : Discutez avec d'autres utilisateurs du site en cliquant sur leur nom dans les paramètres
  • Actions personnelles : Copiez, modifiez ou supprimez vos commentaires
  • Badge de notification : Voyez le nombre de commentaires d'un coup d'œil
  • Temps réel : Synchronisation instantanée avec l'équipe

Collaboration

  • Utilisateurs en ligne : Voyez qui est connecté sur le site et vos projets
  • Chat privé : Cliquez sur un utilisateur connecté pour lui envoyer un message
  • Panel discussions : Accédez à toutes vos conversations via le bouton "Discussions"
  • Notifications : Badge rouge pour les nouveaux messages non lus
  • Temps réel : Messages et statuts synchronisés instantanément

Personnalisation

  • 4 thèmes disponibles : Clair, Sombre, Dragon Vert, Sakura Rose
  • Affichage allégé : Mode simplifié pour se concentrer sur l'essentiel
  • Animation d'arrière-plan : Activez/désactivez les bulles flottantes
  • Préférences sauvegardées : Vos réglages persistent entre les sessions
  • Mode responsive : Interface adaptée à tous les écrans

Gestion de la connectivité

  • Mode hors ligne : Travaillez même sans connexion internet
  • Synchronisation automatique : Vos données se synchronisent au retour de la connexion
  • Indicateur de statut : Voyez votre état de connexion en temps réel
  • Bouton reconnexion : Forcez la reconnexion si nécessaire
  • Sauvegarde locale : Vos modifications sont toujours préservées

Conseils d'utilisation

  • Dates stratégiques : Définissez des échéances réalistes pour un meilleur suivi
  • Descriptions claires : Détaillez vos objectifs pour faciliter la collaboration
  • Statut suspendu : Utilisez-le pour les objectifs en pause temporaire
  • Commentaires réguliers : Tenez l'équipe informée de vos avancées

Besoin d'aide supplémentaire ?

Cette interface intuitive est conçue pour être simple d'utilisation. Explorez les différentes fonctionnalités et n'hésitez pas à expérimenter !
